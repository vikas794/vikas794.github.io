## 🌟 Overview
A high-performance, modern developer portfolio showcasing 4+ years of expertise in building enterprise-grade backend systems. This site features a sleek, responsive design with a focus on visual excellence and optimized performance for recruiters and technical stakeholders.

## 🛠️ Tech Stack
- **Frontend Core**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Logic & Structure**: TypeScript + JSX
- **Design & Styling**: 
  - **Vanilla CSS** with a custom design system of 50+ semantic tokens
  - **Lucide React** for consistent, crisp iconography
- **Animation Engine**: [Motion](https://motion.dev/) (formerly Framer Motion) for fluid, interactive transitions
- **Infrastructure**: [GitHub Pages](https://pages.github.com/) for automated CI/CD deployment

## 🎯 Key Features
- **Performance Optimized**: Implemented `React.lazy` and an intelligent **Background Prefetching** strategy to ensure near-instant section loads after the initial page interact.
- **Micro-interactions**: Custom cursor tracking, typing headers, and dynamic "code-window" logic for a premium feel.
- **Interactive Credentials**: A secure verification gateway for certifications from Google Skills and Credly, specifically engineered to bypass iframe security restrictions.
- **Dark/Light Mode**: Fully responsive, accessible theme engine that syncs all design tokens, including code-syntax highlighting.
- **Mobile First**: 100% responsive architecture with standard industry breakpoints (synchronizing CSS with Tailwind's 768px `md` standard).

## 🚀 Local Development
To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/vikas794/vikas794.github.io.git
   cd vikas794.github.io
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Launch the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
